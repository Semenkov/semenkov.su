Шаблоны (thems) для блога Semenkov.su. Используются статаческим генератором сайтов Pelican.

Ссылка на конкретный применяемый шаблон стоит в конфигурационном файле pelican_local_config.py

THEME = "./pelican-themes/blue"

Схема bootstrap 3 использует дополнительно папку ./content/extra для кастомизированных файлов CSS, чтобы не портить дефалтовые CSS схем.
